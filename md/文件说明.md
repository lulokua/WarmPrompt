## 📜 JS 模块说明

### `core.js` - 核心逻辑 (~400 行)
**职责：** 整体流程控制和状态管理

- **DOM 元素获取**：主容器、输入框、按钮、各选择器容器
- **状态对象 `userData`**：存储用户的所有选择数据
- **颜色方案 `colorSchemes`**：预定义的配色方案
- **事件监听器**：输入、点击、触摸事件
- **核心函数**：
  - `applyBackground()` - 应用背景样式
  - `selectColor()` - 选择颜色主题
  - `submitName()` - 提交当前步骤
  - `goBack()` - 返回上一步
  - `transitionToStep()` - 步骤切换动画
  - `updateStep0UI()` ~ `updateStep6UI()` - 各步骤 UI 更新
  - `showFinalGreeting()` - 显示最终问候语

### `music.js` - 音乐选择器逻辑 (~350 行)
**职责：** 音乐搜索、获取和播放

- **API 配置**：QQ 音乐和网易云音乐 API
- **DOM 元素**：音乐输入框、搜索按钮、结果列表
- **核心函数**：
  - `handleMusicFetch()` - 处理音乐搜索/获取
  - `requestQqMusic()` - 请求 QQ 音乐 API
  - `requestNeteaseMusic()` - 请求网易云 API
  - `normalizeQqMusicData()` / `normalizeNeteaseMusicData()` - 数据标准化
  - `renderMusicResults()` - 渲染搜索结果
  - `renderMusicCard()` - 渲染选中的音乐卡片
  - `setMusicSource()` - 切换音乐来源
  - `updateStep5UI()` - 步骤 5 UI 更新

### `donation.js` - 捐款流程逻辑 (~130 行)
**职责：** QQ 音乐付费功能的捐款引导

- **模态框元素**：QQ音乐提示、悲伤挽留、捐款二维码
- **解锁状态**：`qqMusicUnlocked` 控制 QQ 音乐访问
- **核心函数**：
  - `showQqMusicModal()` / `hideQqMusicModal()` - QQ音乐提示
  - `showSadModal()` / `hideSadModal()` - 悲伤挽留弹窗
  - `showDonateModal()` / `hideDonateModal()` - 捐款弹窗
  - `setDonateType()` - 切换微信/支付宝
  - `downloadQrCode()` - 下载二维码并解锁

### `message.js` - 消息选择器逻辑 (~120 行)
**职责：** 暖心句子选择和自定义输入

- **官方句子库**：`officialQuotes` 数组
- **输入验证**：最多 40 行，每行不超过 6 个字
- **核心函数**：
  - `updateStep7UI()` - 步骤 7 UI 更新
  - `updateMessageModeUI()` - 切换官方/自定义模式
  - `pickRandomQuote()` - 随机选择官方句子
  - `validateCustomMessage()` - 验证自定义输入
