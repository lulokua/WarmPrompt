# 项目目录说明

> 目的：简单说明每个文件夹/文件的用途，方便快速定位。

## 目录（含子目录）

| 路径 | 说明 |
| --- | --- |
| `/md` | 项目文档目录 |
| `/md/项目目录.md` | 本文件，记录目录用途说明 |
| `/api` | 后端接口处理（AI/音乐/管理员/账号等） |
| `/api/log` | 运行日志或调试日志存放（当前为空） |
| `/api/account.js` | 账号相关接口 |
| `/api/admin.js` | 管理员相关接口 |
| `/api/ai.js` | AI 接口与环境变量加载 |
| `/api/netease.js` | 网易云音乐接口 |
| `/api/qqmusic.js` | QQ 音乐接口 |
| `/api/问题.md` | 问题记录/备注 |
| `/public` | 前端静态资源 |
| `/public/404.html` | 404 页面 |
| `/public/index.html` | 首页 |
| `/public/styles.css` | 主站样式 |
| `/public/app.js` | 主站脚本 |
| `/public/vip-styles.css` | VIP 样式 |
| `/public/admini` | 管理端页面 |
| `/public/admini/index.html` | 管理端入口页面 |
| `/public/admini/styles.css` | 管理端样式 |
| `/public/admini/app.js` | 管理端脚本 |
| `/public/admini/dashboard` | 管理端仪表盘 |
| `/public/admini/dashboard/index.html` | 仪表盘页面 |
| `/public/admini/dashboard/styles.css` | 仪表盘样式 |
| `/public/admini/dashboard/app.js` | 仪表盘脚本 |
| `/public/log_in` | 登录页面 |
| `/public/log_in/index.html` | 登录页 |
| `/public/log_in/styles.css` | 登录样式 |
| `/public/log_in/app.js` | 登录脚本 |
| `/public/others` | 其他页面 |
| `/public/others/index.html` | 其他页 |
| `/public/others/styles.css` | 其他页样式 |
| `/public/others/app.js` | 其他页脚本 |
| `/public/self` | 预留/个人页面（当前为空） |
| `/public/font` | 字体文件（MiSans） |
| `/public/font/MiSans-Bold.woff2` | MiSans 字体 |
| `/sql` | 数据库文件（表结构/初始化数据） |
| `/sql/accounts.sql` | 账号相关表结构/数据 |
| `/node_modules` | 依赖包 |
| `/.vscode` | 编辑器配置 |
| `/.codex` | Codex 本地技能配置 |
| `/.git` | Git 版本管理数据 |
| `/.env` | 环境变量配置 |
| `/.gitignore` | Git 忽略规则 |
| `/package.json` | 项目依赖与脚本 |
| `/package-lock.json` | 依赖锁定 |
| `/server.js` | Node.js 服务入口（静态文件 + API 路由） |
